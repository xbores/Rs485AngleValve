# RS485 角行程球阀(电动调节阀)控制器 使用说明书

> 适用固件:STM32F030 平台,Keil 工程输出 `PB331.hex`
> 通讯:RS485 / Modbus-RTU
> 文档版本:2026-06-28

---

## 1. 产品概述

本产品是一款带 RS485 / Modbus-RTU 通讯的**电动角行程球阀控制器**,内置角度反馈、流量计量与 4~20mA 模拟量输入,可对球阀开度进行远程设定与闭环调节。提供 **四种控制模式**:

| 模式 | 名称 | 说明 |
|------|------|------|
| 0 | 角度模式 | 按目标角度 / 开度定位(角度反馈闭环) |
| 1 | 流量模式 | 按目标瞬时流量 PID 调节 |
| 2 | 压力模式 | 按 4~20mA 输入(压力)目标值 PID 调节 |
| 3 | 给定模式 | 按外部给定值与目标值偏差 PID 调节(极性可选) |

核心功能:

- 角度反馈(电位器,ADC 通道 1),上电自动标定全关 / 全开两端位置与全行程时间。
- 流量脉冲计量:瞬时流量(L/H,可选卡尔曼滤波)+ 累计流量(L,断电保存于 EEPROM)。
- 4~20mA 模拟量输入(ADC 通道 0),可作压力 / 通用模拟量,支持两点标定与偏移。
- 全部参数掉电保存于片内 Flash;通讯地址、波特率可远程修改。
- 4 秒独立看门狗,异常自动复位。

## 2. 硬件接口

| 功能 | 引脚 | 说明 |
|------|------|------|
| RS485 (USART1) | PA2 = TX,PA3 = RX | 8 数据位 / 无校验 / 1 停止位(8N1) |
| 电机正转(开阀) | PA6 | H 桥三态控制 |
| 电机反转(关阀) | PA5 | H 桥三态控制 |
| 电机转动反馈 | PA7(输入) | 检测电机是否在转(标定 / 到位判断) |
| 流量脉冲输入 | PA4(外部中断,下降沿计数) | 接流量计脉冲输出 |
| 模拟量输入 0 | PA0(ADC) | 4~20mA / 压力输入 |
| 角度反馈输入 | PA1(ADC) | 阀位电位器 |
| EEPROM | I2C(AT24Cxx) | 保存累计流量 |

> 电机控制为三态:**正转 / 反转 / 刹车**。刹车时两路同时驱动并保持 200ms 后释放。

## 3. 通讯协议

- **协议**:Modbus-RTU
- **默认从站地址**:`13`
- **默认波特率**:`9600`(波特率索引 3)
- **数据格式**:8N1(8 数据位,无校验,1 停止位)
- **寄存器**:16 位保持寄存器
- **支持的功能码**:

| 功能码 | 含义 | 说明 |
|--------|------|------|
| `0x03` | 读保持寄存器 | 一次最多读 120 个寄存器(地址 + 数量 ≤ 120) |
| `0x06` | 写单个寄存器 | 主要写入方式 |
| `0x10` | 写多个寄存器 | 仅用于"清零累计流量"(写寄存器 9 起 2 个) |

### 通讯行为说明

- **广播地址 255**:所有从站执行命令但**不返回应答**(避免总线冲突)。
- **CRC 校验错**:静默丢弃,**不应答**(符合 Modbus 规范)。
- **地址不匹配**:不应答。
- **非法功能码 / 非法寄存器地址 / 读取数量越界**:返回异常码(0x80 | 功能码)。
- 帧间隔由内部定时器按波特率自动判定一帧结束。
- 修改波特率(寄存器 12)后**立即生效**。

## 5. 控制模式详解

控制模式由**写入相应寄存器自动切换**,无需单独设置模式寄存器。寄存器 6(读)随时反映当前模式。

### 5.1 角度模式(模式 0)
- 触发:写寄存器 0(目标角度)或寄存器 1(目标开度)。
- 行为:控制器读取角度反馈(寄存器 21),换算为当前角度 / 开度,驱动电机定位到目标。
- 死区:目标与当前角度之差在 ±0.5°(±5)以内视为到位,电机刹车停止。
- 反向保护:不会从正转直接切到反转,先刹停再换向。

### 5.2 流量模式(模式 1)
- 触发:写寄存器 7(目标瞬时流量,L/H)。
- 行为:每个流量采样周期(≥500ms)计算一次增量式 PID,按结果驱动电机一段时间使当前流量(寄存器 8)趋近目标。
- 瞬时流量公式:`流量(L/H) = 脉冲数 × 3,600,000 ÷ (流量计系数 × 采样周期ms)`。

### 5.3 压力模式(模式 2)
- 触发:写寄存器 25(目标压力)。
- 行为:基于 4~20mA 输入(寄存器 26 当前压力)做增量式 PID 调节。
- 压力换算:ADC0 → 电流百分比;`4mA(=400) → 4mA对应值(reg22)`,`20mA(=2000) → 20mA对应值(reg23)`,中间线性插值并加偏移量(reg24)。低于 4mA 取下限值,高于 20mA 取上限值。

### 5.4 给定模式(模式 3)
- 触发:写寄存器 29(当前给定值,即外部反馈量)。
- 行为:PID 使**当前给定值(reg29)跟踪目标给定值(reg28)**,调节方向由极性(reg31)决定。
- 适用于把任意外部测量量(由主站周期写入 reg29)闭环到设定值(reg28)的场合。

> PID 输出限幅为 ±1000,实际换算为电机驱动时间(毫秒);三种 PID 模式分别用最大流量 / 最大压力 / 最大给定值作归一化分母。

### 5.5 模糊 PID(可选,作用于流量/压力/给定模式)
流量、压力、给定三种模式支持**模糊自整定 PID**:以误差 E 和误差变化 EC 为输入,经模糊规则在线调整 Kp/Ki/Kd,改善大偏差快速响应、小偏差平稳收敛的折中。

- **默认关闭**(寄存器 32 = 0),此时使用经典增量 PID,与旧版本行为一致;需要时写寄存器 32 = 1 启用。
- 启用后,P/I/D(寄存器 16/17/18)仍是基础值,模糊算法在其基础上动态加减。
- 整定行为由 4 个参数控制:
  - **Kec 倍数(寄存器 33)**:误差变化的灵敏度,默认 4。偏大更跟随变化,偏小更平稳。
  - **Kp/Ki/Kd 整定强度%(寄存器 34/35/36)**:各增益的浮动幅度(基础值的百分比),默认 30%。设 0 则该增益不参与模糊整定。
- 量化因子 Ke 由固件自动取 `3 / 满量程`(满量程即对应模式的最大流量/最大压力/最大给定值),无需手动设置。
- 调参建议:先用经典 PID 整定好基础 P/I/D,再启用模糊并从默认 30% 起逐步调整整定强度,观察超调与稳态。

## 6. 上电与标定流程

1. **首次上电 / 恢复出厂后**(寄存器 3 或 4 为 0 时)自动标定:
   - 电机全关到底 → 记录"关闭 ADC 值"(reg4);
   - 电机全开到底 → 记录"开启 ADC 值"(reg3),并把全行程时间写入"最大行程"(reg15,单位秒);
   - 结果存入 Flash,之后上电不再重复标定。
2. **按初始状态动作**(寄存器 5):
   - `0` = 上电后关闭阀门;
   - `1` = 上电后开启阀门;
   - `2` = 保持上次位置(默认)。
3. 进入工作循环,等待 Modbus 指令。

> 如需重新标定,可写寄存器 6 恢复出厂(reg3/4 清零),下次上电将自动重新标定。

## 7. 特殊操作

| 操作 | 方法 |
|------|------|
| 恢复出厂设置 | 向**寄存器 6** 写任意值 → 清空参数并立即复位重启 |
| 清零累计流量 | 向**寄存器 9**(或 10)写任意值;或用功能码 0x10 写寄存器 9 起 2 个寄存器 |
| 修改从站地址 | 写寄存器 11(1~254),掉电保存 |
| 修改波特率 | 写寄存器 12(索引 0~6),立即生效并保存 |
| 广播设置 | 用从站地址 255 发送写命令,所有阀门同时执行且不应答 |

## 8. 通讯报文示例

从站地址以默认 `13`(0x0D)为例;报文为十六进制,末尾两字节为 CRC16(低字节在前)。

### 读取类(功能码 0x03)

| 操作 | 报文 |
|------|------|
| 读当前角度(reg0,1 个) | `0D 03 00 00 00 01 84 C6` |
| 读概览(reg0 起 8 个) | `0D 03 00 00 00 08 44 C0` |
| 读累计流量(reg9 起 2 个) | `0D 03 00 09 00 02 14 C5` |
| 读 ADC 实时值(reg20 起 2 个) | `0D 03 00 14 00 02 84 C3` |

### 写入类(功能码 0x06)

| 操作 | 报文 |
|------|------|
| 设目标角度 45.0°(reg0 = 450) | `0D 06 00 00 01 C2 09 07` |
| 设目标开度 50.0%(reg1 = 500) | `0D 06 00 01 01 F4 D8 D1` |
| 进入流量模式,目标 500 L/H(reg7 = 500) | `0D 06 00 07 01 F4 38 D0` |
| 进入压力模式,目标 0.500MPa(reg25 = 500) | `0D 06 00 19 01 F4 58 D6` |
| 清零累计流量(reg9) | `0D 06 00 09 00 00 59 04` |
| 修改从站地址为 20(reg11 = 20) | `0D 06 00 0B 00 14 F8 CB` |
| 修改波特率为 19200(reg12 = 4) | `0D 06 00 0C 00 04 48 C6` |
| 恢复出厂设置(reg6) | `0D 06 00 06 00 01 A8 C7` |

> 修改地址 / 波特率后,后续通讯需使用新的地址 / 波特率。

## 10. 注意事项

1. 修改通讯地址或波特率后请记录新值,否则需恢复出厂找回默认 13 / 9600。
2. "流量计系数 / 最大流量 / 最大压力 / 最大给定值"作为运算分母,固件已禁止写 0,但请按实际填写有效值。
3. 写寄存器 6 会**立即清空全部参数并重启**,请谨慎,勿误用于读取控制方式(读寄存器 6 才是查询模式)。
4. 标定与上电动作期间电机会自动运行,请确保阀门机械行程畅通、限位正常。
5. 多台设备共用 RS485 总线时,务必每台分配唯一从站地址。
